# URL escape code
```
퍼센트 인코딩이라고도하며, URL encoding에 사용된다. 

%뒤에 아스키 코드의 hex(16진수)값을 붙여준 것. URL은 스페이스(space)를 포함할 수 없기 때문에, 스페이스는 %20으로 대체되는 것이 보통
```

# EX
```
http://namu.wiki/w/나무위키
-> 
http://namu.wiki/w/%EB%82%98%EB%AC%B4%EC%9C%84%ED%82%A4
```
