# 멀티 프로세스 vs 멀티 쓰레드

멀티 프로세싱을 할 때 Context Switching에서의 오버헤드 발생가능성

(Context Switching : CPU에서 여러 프로세스를 돌아가면서 작업을 처리하는 과정)

멀티 쓰레드 : 하나의 프로그램에서 여러작업 가능, 처리비용 감소, 자원의 효율성 증대 but 주의 깊은 설계와 까다로운 디버깅
